# API Endpoints

## Audio
- `POST /audio-assets` - Create audio asset
- `GET /audio-assets` - List audio assets
- `GET /audio-assets/:id` - Get audio asset by ID
- `PUT /audio-assets/:id` - Update audio asset
- `DELETE /audio-assets/:id` - Delete audio asset

## Video
- `POST /video-assets` - Create video asset
- `GET /video-assets` - List video assets
- `GET /video-assets/:id` - Get video asset by ID
- `PUT /video-assets/:id` - Update video asset
- `DELETE /video-assets/:id` - Delete video asset

## User
- `POST /users` - Create user
- `GET /users` - List users
- `GET /users/:id` - Get user by ID
- `PUT /users/:id` - Update user
- `DELETE /users/:id` - Delete user
