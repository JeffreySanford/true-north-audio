# Backend Overview

## Technologies
- NestJS
- MongoDB (Mongoose ODM)
- WebSocket streaming

## Models
- Genre
- VocalFeature
- User (with roles)
- AudioAsset
- VideoAsset (planned)

- No OAuth yet

## Extensibility
- Easy to add new models and endpoints
